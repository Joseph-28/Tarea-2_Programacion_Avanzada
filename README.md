# Control de Tareas Pendientes - Tarea 2

Esta aplicación web ha sido desarrollada como parte de la Tarea No. 1 del curso 03101 | Programación Avanzada en Web de la Universidad Estatal a Distancia (UNED), correspondiente al II Cuatrimestre de 2026 por Joseph González Herrera.

La aplicación fue desarrollada con el fin de registrar, consultar, editar, eliminar y gestionar tareas pendientes. 

## Tecnologías Utilizadas
- ASP.NET MVC 5.
- C#.
- Razor Views.
- HTML5.
- CSS3.
- JavaScript.
- jQuery.
- AJAX.
- Data Annotations.
- Session State.

## Instrucciones para Ejecutar el Proyecto
## Requisitos Previos
- Visual Studio 2022 o superior.
- .NET Framework compatible con ASP.NET MVC 5.
- Navegador web actualizado.
- Pasos de Ejecución.

## Instrucciones de Instalación y Ejecución
1. Clonar o descargar el repositorio: https://github.com/Joseph-28/Tarea-2_Programacion_Avanzada

## Notas
- La aplicación utiliza Session para almacenar los datos.
- No requiere base de datos para su funcionamiento.
- Los datos se mantienen únicamente durante la sesión activa del usuario.
